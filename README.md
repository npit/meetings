# Tool to produce meet schedules from a list of per-person restrictions.

Launch:

```
python sched.py schedulename
```

Add restrictions:

```
jon can mon after 14
mary cant fri before 20
mary cant wed 14-15
```

View information:

```
s[how] 
s p[eople]
s r[estrictions]
s t[ables]
s mary
```

Clear restrictions or completely delete.

```
jon clear
jon delete
```
